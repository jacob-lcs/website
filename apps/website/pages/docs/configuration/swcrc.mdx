# 配置 SWC

SWC 可以通过 `.swcrc` 文件进行配置。

## 编译

SWC 的编译功能开箱即用，无需自定义。你也可以选择覆盖默认配置。以下是默认配置：

```jsonc filename=".swcrc" copy
{
  "$schema": "https://swc.rs/schema.json",
  "jsc": {
    "parser": {
      "syntax": "ecmascript",
      "jsx": false,
      "dynamicImport": false,
      "privateMethod": false,
      "functionBind": false,
      "exportDefaultFrom": false,
      "exportNamespaceFrom": false,
      "decorators": false,
      "decoratorsBeforeExport": false,
      "topLevelAwait": false,
      "importMeta": false
    },
    "transform": null,
    "target": "es5",
    "loose": false,
    "externalHelpers": false,
    // 需要 v1.2.50 或更高版本，并且目标版本需要是 es2016 或更高。
    "keepClassNames": false
  },
  "minify": false
}
```

了解更多关于[配置编译](/docs/configuration/compilation)的信息。

## 支持的浏览器

从 `v1.1.10` 开始，你可以使用 `browserslist` 自动配置支持的浏览器。

### 使用方法

首先，安装 `browserslist`。然后，更新你的 `.swcrc` 文件：

```json filename=".swcrc" copy
{
  "env": {
    "targets": {
      "chrome": "79"
    },
    "mode": "entry",
    "coreJs": "3.22"
  }
}
```

了解更多关于[配置支持的浏览器](/docs/configuration/supported-browsers)的信息。

## 代码压缩

从 `v1.2.67` 开始，你可以通过在 `.swcrc` 文件中启用 `minify` 来配置 SWC 压缩代码：

```json filename=".swcrc" copy
{
  "minify": true
}
```

了解更多关于[配置 JavaScript 压缩器](/docs/configuration/minification)的信息。
