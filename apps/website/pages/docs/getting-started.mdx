# 入门

## 安装

使用 [Playground](/playground) 是最简单的方法。

否则，运行以下命令下载预构建的二进制文件：

<Tabs items={['pnpm', 'npm', 'yarn']}>
  <Tab>
    ```bash
    pnpm add -D @swc/cli @swc/core
    ```
  </Tab>
    <Tab>
    ```bash
    npm i -D @swc/cli @swc/core
    ```
  </Tab>
    <Tab>
    ```bash
    yarn add -D @swc/cli @swc/core
    ```
  </Tab>
</Tabs>

然后，你可以转译你的第一个文件并输出到 `stdout`:

```bash
npx swc ./file.js
```

## 支持的二进制文件

SWC 可以作为预构建的二进制文件下载和使用，或者从源代码构建。目前提供以下二进制文件：

- Mac (Apple Silicon)
- Mac (x64)
- Linux (x86_64)
- Linux (aarch64)
- Linux (armv7)
- Alpine Linux (also install `@swc/core-linux-musl`)
- Android (aarch64)
- Windows (win32-x64)
- Windows (ia32)
